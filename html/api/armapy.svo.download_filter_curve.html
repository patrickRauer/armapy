download_filter_curve¶

armapy.svo.download_filter_curve(telescope, instrument, band, path, file_type='VO')¶

Downloads a filter transmission curve from the SVO web page.

Parameters:
  • telescope (str) – The name of the telescope/ satellite
  • instrument (str) – The name of the instrument (if the telescope has only one instrument, the name of the instrument is equal to the telescope name)
  • band (str) – The name of the band
  • path (str) – The path to the save place
  • file_type (str) – The required file type, available formats are ‘ascii’ or ‘VO’
