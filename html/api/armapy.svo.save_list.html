save_list¶

armapy.svo.save_list(surveys, path='./filter_list_svo.fits', update=False)¶

Saves the list of the available SVO filters

Parameters:
  • surveys (list) – The output of get_svo_filter_list()
  • path (str) – The path to the save place
  • update (bool) – True if the current file should be overwrite, else false.
Returns:

A table with all the filters.
Return type:

astropy.table.Table
