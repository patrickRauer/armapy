Band¶

class armapy.mag_calc.Band(band=None, wavelength=None, response=None, smt='slinear')¶

Bases: armapy.mag_calc.Spectrum

Band passes photometric response curves

Parameters:
  • band (str) – indicates the name of a filter in the internal list (see show_filters function) or a filename of a file containing two columns, one with the wavelength and one with the normalized response
  • smt (str) – indicates the type of smoothing to perform on the response curve. Uses same names as Scipy’s interp1d (‘linear’, ‘nearest’, ‘zero’, ‘slinear’, ‘quadratic’, ‘cubic’) Default is slinear
