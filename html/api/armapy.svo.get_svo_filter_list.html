get_svo_filter_list¶

armapy.svo.get_svo_filter_list(path='', update=False)¶

Creates a complete list of all filter-curves, which are available on the SVO web page.

Returns:A table with the telescope names, instrument names and the filter names
Return type:astropy.table.Table
